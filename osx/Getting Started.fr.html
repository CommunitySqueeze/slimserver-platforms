Installation

Pour installer le Squeezebox Server, cliquez deux fois sur le fichier d'installation et suivez les instructions à l'écran. Le programme d'installation vous avertira une fois l'installation terminée. Le Squeezebox Server pour Mac OS X est installé dans le volet Préférences Système. Vous pouvez utiliser ce volet pour effectuer les actions suivantes.

  • Démarrer et arrêter le Squeezebox Server manuellement, ou le configurer pour qu'il démarre automatiquement au démarrage de l'ordinateur ou à l'ouverture de la session.
  • Démarrer l'interface Web du Squeezebox Server.

Lors de la première exécution, un assistant vous guide dans la procédure de configuration du Squeezebox Server pour votre système.

Si vous effectuez une mise à niveau d'une ancienne version du SlimServer vers le Squeezebox Server, les paramètres du SlimServer seront conservés. Cependant, en raison des modifications apportées à l'architecture des extras, vous devrez télécharger et installer des versions mises à jour. Les réglages spécifiques aux extras ne seront pas conservés automatiquement. Pour en savoir plus sur les extras, rendez-vous sur notre wiki.

Vous pouvez obtenir des informations sur l'utilisation du serveur dans l'aide du Squeezebox Server en sélectionnant l'icône Aide dans la partie inférieure gauche de l'interface Web.

Mise à jour du micrologiciel

Si vous possédez une Squeezebox ou un Transporter, un message vous demandant de mettre à jour le micrologiciel peut s'afficher lors de la première connexion au Squeezebox Server. Maintenez le bouton Brightness de la télécommande enfoncé jusqu'à ce que la mise à jour démarre.

Saisie d'un mot de passe WPA à l'aide de la télécommande

La section relative aux mots de passe WPA dans certaines versions des guides de l'utilisateur de la Squeezebox et de Transporter contient une erreur. Selon ces guides, vous devez utiliser le bouton PLAY pour insérer un caractère dans la chaîne du mot de passe et le bouton ADD pour en supprimer un. Il faut en réalité comprendre le contraire. Le bouton ADD permet d'insérer un caractère, et le bouton PLAY d'en supprimer un.

Prise en charge de la bibliothèque d'iTunes

Si vous utiliser Apple iTunes pour gérer votre musique, le Squeezebox Server importe les informations de la bibliothèque automatiquement. Si vous utilisez iTunes pour ajouter des morceaux à la bibliothèque ou pour en supprimer, le Squeezebox Server détecte les modifications apportées et analyse à nouveau la bibliothèque. Vous pouvez définir l'intervalle entre les vérifications à partir de l'onglet iTunes sous Réglages dans l'interface Web du Squeezebox Server.

Si vous ne souhaitez pas que le Squeezebox Server utilise la base de données iTunes, vous devez modifier le réglage dans l'onglet iTunes. Vous devez spécifier le dossier dans lequel votre musique et vos listes de lecture sont stockées, puis démarrer une analyse en sélectionnant l'option Analyser dans l'onglet Réglages de base.

A propos des morceaux achetés sur iTunes Music Store

Apple utilise le système de gestion numérique des droits (DRM) FairPlay sur la plupart des morceaux vendus sur iTunes Music Store. Les droits des utilisateurs sont par conséquent limités sur ces fichiers. Apple n'autorise pas la lecture de tels morceaux sur un périphérique non fourni par Apple (à l'exception de l'ordinateur à l'aide duquel le morceau a été acheté). Par conséquent, vous ne pourrez pas lire ces morceaux à l'aide de la Squeezebox ou de Transporter en raison de ce système.

Certains morceaux de la bibliothèque iTunes ne s'affichent pas dans le Squeezebox Server ou ne peuvent être ouverts

Si vous avez déplacé les morceaux, les informations sur la bibliothèque iTunes risquent d'être incorrectes. Vous devez spécifier le dossier correct dans iTunes.

Pour ce faire, accédez aux préférences, puis cliquez sur Avancé. Spécifiez le chemin d'accès correct au dossier iTunes Music. S'il est correct, essayez de spécifier un dossier différent, puis spécifiez à nouveau le dossier correct. iTunes met ensuite à jour les chemins d'accès à tous les morceaux et exporte ces informations de manière adéquate. Cette procédure ne déplace aucun morceau mais force iTunes à analyser la bibliothèque musicale et à mettre à jour le chemin d'accès aux morceaux.

Si le problème persiste, la confusion peut provenir des modifications apportées aux options Maintenir organisé le dossier iTunes Music et Créer des noms de fichier avec le numéro de piste. Modifiez ces options pour qu'iTunes mette à jour les informations sur les fichiers musicaux. Notez que le fait de laisser iTunes gérer les fichiers musicaux peut impliquer l'attribution d'un nouveau nom aux fichiers ou leur déplacement.

Diffusion de musique sur Internet

Le Squeezebox Server a été conçu pour diffuser des morceaux au format MP3 vers une platine Squeezebox, Transporter ou SLIMP3 en réseau. Il permet également de diffuser vos morceaux sur Internet pour les lecteurs prenant en charge la diffusion de musique à ce format.

Installez le Squeezebox Server et vérifiez que l'ordinateur sur lequel le serveur est installé est connecté à Internet. Si ce n'est pas le cas, vous devez ouvrir le port 9000 sur le routeur vers l'ordinateur sur lequel le Squeezebox Server est installé. Reportez-vous à la documentation du routeur pour en savoir plus. Procédez ensuite comme suit.

  • Utilisez le logiciel de lecture de MP3 pour ouvrir un flux à l'aide de l'URL suivante: http://localhost:9000/stream.mp3. Remplacez localhost par l'adresse IP de l'ordinateur du Squeezebox Server. Si vous utilisez un mot de passe pour le Squeezebox Server, vous devez spécifier une URL du type http://nom_d'utilisateur:mot_de_passe@localhost:9000/stream.mp3. Certains logiciels de lecture de MP3 vous invitent à spécifier un nom d'utilisateur et un mot de passe.
  • Ouvrez l'interface Web du Squeezebox Server exécuté sur l'ordinateur distant en ouvrant le navigateur à la page http://localhost:9000. Remplacez localhost par l'adresse IP de l'ordinateur distant. Vous pourrez apercevoir une platine correspondant à l'adresse IP de l'ordinateur doté du logiciel de lecture MP3.
  • Vous pouvez utiliser le volet de gauche de l'interface Web du Squeezebox Server pour parcourir les fichiers et les listes de lecture et effectuer vos sélections. Lorsque vous sélectionnez un morceau, celui-ci s'affiche dans le volet de gauche de l'interface Web.
  • Cliquez sur Lecture dans le volet de droite de l'interface pour lancer la lecture du morceau.
  • Le logiciel de lecture de MP3 commence à diffuser le morceau après quelques secondes. Ce délai est dû à la mise en tampon du morceau par le logiciel de lecture.
  • Utilisez l'interface Web du Squeezebox Server sur l'ordinateur distant pour changer de morceau.
