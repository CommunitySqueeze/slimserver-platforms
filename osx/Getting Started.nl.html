Installatie

Dubbelklik op het Squeezebox Server-installatieprogramma en volg de instructies op het scherm om Squeezebox Server te installeren. Het programma vertelt je wanneer de installatie voltooid is. Squeezebox Server voor Mac OS X wordt geïnstalleerd als Systeemvoorkeuren-paneel. Gebruik dit paneel om de volgende taken uit te voeren:

  • Squeezebox Server handmatig starten en stoppen, of automatisch laten starten wanneer het systeem opstart of je je aanmeldt
  • De Squeezebox Server-webinterface starten

Wanneer je de webinterface voor het eerst uitvoert, leidt een wizard je door de Squeezebox Server-configuratie voor je systeem.

Als je een bestaande installatie van SlimServer naar Squeezebox Server opwaardeert, blijven je SlimServer-instellingen bewaard. Vanwege wijzigingen in de plugin-architectuur moet je echter bijgewerkte versies van je plug-ins downloaden en installeren. Specifieke instellingen voor plug-ins worden niet automatisch bewaard. Bezoek onze wiki voor meer informatie over plug-ins.

Je kunt informatie over Squeezebox Server oproepen door het Help-pictogram in de linkerbenedenhoek van de Squeezebox Server-webinterface te selecteren.

Wat is er nieuw in 7.3?

Er is ook een volledige lijst met veranderingen.

Firmware-update

Als je een Squeezebox of Transporter hebt en deze voor het eerst met Squeezebox Server verbindt, word je misschien gevraagd je firmware te updaten. Houd de knop Brightness op de afstandsbediening ingedrukt tot de firmware-update begint.

Een WPA-wachtwoord met de afstandsbediening invoeren

In sommige handleidingen van de Squeezebox en Transporter staat een fout in de sectie over WPA-wachtwoorden. Volgens de handleiding zou je de knop PLAY moeten gebruiken om een teken voor het wachtwoord in te voeren en ADD om een teken te verwijderen, maar het is precies andersom: met ADD voeg je een teken in en met PLAY verwijder je een teken.

Support voor iTunes-collectie

Als je Apple's iTunes gebruikt om je muziek te beheren, importeert Squeezebox Server automatisch de gegevens van je iTunes-collectie. Gebruik je iTunes om muziek aan de collectie toe te voegen of eruit te verwijderen, dan detecteert Squeezebox Server de verandering en scant het de informatie over je liedje opnieuw. Je kunt het interval tussen controles instellen op de tab iTunes onder 'Instellingen' in de Squeezebox Server-webinterface.

Als je niet wilt dat Squeezebox Server je iTunes-database gebruikt, moet je de instelling wijzigen op de tab iTunes onder 'Instellingen' in de Squeezebox Server-webinterface. Je moet de map opgeven waarin je muziek en playlists zijn opgeslagen, en vervolgens een scan van je muziek starten door 'Opnieuw scannen' te selecteren op het tabblad Basisinstellingen.

Liedjes die in de iTunes Music Store zijn gekocht

Aangezien Apple een DRM-wrapper (Digital Rights Management) gebruikt rond de meeste liedjes die via de iTunes Music Store worden verkocht, bieden deze bestanden beperkte gebruikersrechten. Eén belangrijk recht dat Apple je niet verleent, is het recht om dat nummer op een muziekapparaat af te spelen dat niet door Apple is geleverd (met uitzondering van de computer waarop je het nummer hebt gekocht). Tot Apple zijn standpunt over het licentiëren of elimineren van FairPlay herziet, kun je deze nummers niet op de Squeezebox of Transporter afspelen.

Sommige liedjes in de iTunes-collectie verschijnen niet in Squeezebox Server of kunnen niet worden geopend

Als je je iTunes-muziekbestanden verplaatst hebt, is het mogelijk dat iTunes onjuiste informatie over de liedjes in zijn collectie exporteert. Je moet ervoor zorgen dat iTunes de juiste map voor je muziekcollectie gebruikt.

Ga in iTunes naar 'Voorkeuren' en klik op 'Geavanceerd'. Als het pad naar de map met je muziekcollectie verkeerd is, moet je dit corrigeren. Is het pad juist, wijzig het dan naar een andere map en vervolgens weer terug naar de juiste map. iTunes werkt de paden naar al je muziek dan bij en zal de gegevens goed exporteren. (Met dit proces wordt je muziek niet verplaatst, maar wordt iTunes gedwongen je muziekcollectie te scannen en de paden naar de liedjes te updaten.)

Als je probleem hiermee niet wordt opgelost, is iTunes misschien in de war geraakt toen je de instelling 'Map iTunes Music bijwerken' of 'Bestandsnamen met tracknummer aanmaken' in iTunes hebt veranderd. Verander deze instellingen en zet ze dan weer terug op de oorspronkelijke instellingen, zodat iTunes de juiste informatie over je muziekbestanden kan bijwerken. Wanneer iTunes je muziekbestanden organiseert, worden de bestanden misschien verplaatst of hernoemd.

Het afspelen streamen via het internet

Squeezebox Server is ontworpen om mp3-muziek naar een Squeezebox-, Transporter- of SLIMP3-netwerkmuziekspeler te streamen. Het kan dezelfde muziek ook over het internet streamen naar een softwarespeler die mp3-streaming ondersteunt.

Installeer eerst Squeezebox Server en zorg dat de computer die als de server moet worden gebruikt, via het internet kan worden bereikt. Als dit niet zo is, moet je poort 9000 op je router naar de computer leiden waarop Squeezebox Server wordt uitgevoerd. (Raadpleeg de documentatie van je router voor instructies.) Daarna doe je het volgende:

  • Gebruik je mp3-spelersoftware om een stream naar de URL te openen: http://localhost:9000/stream.mp3. (Vervang 'localhost' door het IP-adres van de computer waarop Squeezebox Server wordt uitgevoerd.) Als je de wachtwoordbeveiliging van Squeezebox Server gebruikt, moet je een enigszins gewijzigde URL gebruiken, zoals deze: http://username:password@localhost:9000/stream.mp3. Sommige spelers vragen je om je gebruikersnaam en wachtwoord.
  • Start de Squeezebox Server-webinterface die op de externe computer wordt uitgevoerd, door de webpagina http://localhost:9000 te openen (vervang 'localhost' door het IP-adres van de externe computer). Je zult merken dat er een nieuwe 'speler' is die overeenkomt met het IP-adres van de computer waarop de mp3-softwarespeler zich bevindt.
  • Gebruik het linkerpaneel van de Squeezebox Server-webinterface om de muziek die je wilt afspelen, te zoeken en selecteren. Wanneer muziek wordt geselecteerd, verschijnt deze in het rechterpaneel van de webinterface.
  • Klik op 'Afspelen' in het rechterpaneel van de Squeezebox Server-webinterface om de muziek te starten.
  • Na enkele seconden hoor je dat er muziek afgespeeld wordt via de mp3-softwarespeler. (De vertraging wordt veroorzaakt door bufferen in de mp3-spelersoftware.)
  • Gebruik de Squeezebox Server-webinterface op de externe machine om de afgespeelde inhoud te wijzigen.
