Installation

Doppelklicken Sie zum Installieren von SqueezeCenter auf das SqueezeCenter-Installationsprogramm und folgen Sie den Anweisungen auf dem Bildschirm. Das Installationsprogramm meldet, wenn die Installation abgeschlossen ist. SqueezeCenter for Windows wird standardmäßig als Anwendung installiert. Informationen zum Ausführen von SqueezeCenter als Systemdienst finden Sie hier. Doppeklicken Sie auf das SqueezeCenter-Symbol in der Windows-Taskleiste, um die SqueezeCenter-Web-Benutzeroberfläche zu starten.

Wenn Sie die Web-Benutzeroberfläche zum ersten Mal starten, führt Sie ein Assistent durch die Konfiguration von SqueezeCenter für Ihren Computer.

Wenn Sie SlimServer auf SqueezeCenter aufrüsten, bleiben die Einstellungen aus SlimServer erhalten. Allerdings müssen Sie aufgrund der geänderten Plug-In-Architektur aktualisierte Plug-Ins herunterladen und installieren. Plug-In-spezifische Einstellungen bleiben nicht automatisch erhalten. Informationen zu Plug-Ins finden Sie in unserem Wiki:

Informationen zum Einsatz von SqueezeCenter erhalten Sie, wenn Sie auf das Hilfesymbol unten links in der Web-Benutzeroberfläche klicken.

Neue Funktionen in 7.3

SqueezeCenter Version 7.3 enthält bedeutende Änderungen sowie einen brandneuen Namen! Die folgende Liste enthält die wichtigsten Änderungen, die die Anzeige auf dem Bildschirm betreffen oder die Funktionsweise Ihrer Squeezebox oder Ihres Transporter ändern bzw. verbessern. Eine vollständige Liste ist verfügbar.

  • SlimServer wurde im Angleich an die Produktnamen in SqueezeCenter umbenannt.
  • SqueezeCenter hat eine vollständig neue Standardoberfläche. Die Einstellungen befinden sich auf einer Seite mit Registerkarten, was das Konfigurieren des Programms erleichtert. Wenn Sie die "alte" Benutzeroberfläche von SlimServer 6.5.x bevorzugen, können Sie unter http://your server ip:9000/classic auf die Version "Classic" zugreifen.
  • SqueezeNetwork ist jetzt in SqueezeCenter integriert, Sie müssen also nicht mehr zwischen den beiden Anwendungen umschalten, um auf Internetdienste bzw. die lokale Musikbibliothek zuzugreifen. Dienste wie Pandora Radio, Rhapsody oder Slacker Radio können jetzt direkt von einer Squeezebox oder einem Transporter, der mit SqueezeCenter verbunden ist, aufgerufen werden.
  • Die Synchronisierung von mehreren Geräten (Squeezebox oder Transporter) wurde völlig umgestaltet und verbessert.
  • Die Unterstützung SqueezeOS basierender Geräte wurde hinzugefügt. (SqueezeOS wurde bisher "Jive-Software-Plattform" genannt.)
  • Die Plug-In-Architektur wurde umgestaltet, um eine robuste, konsistente Oberfläche zu schaffen. Der Nachteil ist, dass die Plug-Ins aus SlimServer 6.5.x von ihren Entwicklern aktualisiert werden müssen, damit sie mit SqueezeCenter kompatibel sind. Informationen zu SqueezeCenter-Plug-Ins finden Sie hier.

Ausführen von SqueezeCenter als Windows-Dienst

Standardmäßig wird SqueezeCenter jetzt im Anwendungsmodus ausgeführt und beim Anmelden bei Windows gestartet. SqueezeCenter kann so auf Ihre iTunes-Bibliothek, freigegebenen Order oder Netzwerklaufwerke zugreifen, ohne dass sich die Funktionsweise ändert. Wenn Sie SqueezeCenter als Windows-Dienst im Hintergrund ausführen möchten, müssen Sie die Funktion SqueezeCenter-Dienst aktivieren ausführen. Wählen Sie im Windows-Startmenü im SqueezeCenter-Programmordner die Option Startup Manager.

  • Wählen Sie die Option "SqueezeCenter automatisch beim Systemstart starten.", um SqueezeCenter als Dienst zu installieren.
  • Wählen Sie die Option "SqueezeCenter beim Anmelden starten", um SqueezeCenter als Anwendung auszuführen (Standardeinstellung).

Wenn Sie SqueezeCenter als Dienst im Hintergrund ausführen, müssen Sie Windows-Benutzername und Kennwort eingeben, damit SqueezeCenter auf Ihre iTunes-Bibliothek, freigegebenen Ordner oder Netzwerklaufwerke zugreifen kann. Klicken Sie auf Weiter, um die Änderungen zu übernehmen und die Anwendung zu beenden. Sie müssen Windows neu starten, um den Vorgang abzuschließen.

Aktualisieren der Firmware

Wenn Sie eine Squeezebox oder einen Transporter haben, werden Sie eventuell aufgefordert, die Firmware zu aktualisieren, wenn zum ersten Mal eine Verbindung zu SqueezeCenter hergestellt wird. Halten Sie die Brightness-Taste auf der Fernbedienung gedrückt, bis das Aktualisieren der Firmware gestartet wird.

Eingeben eines WPA-Kennworts über die Fernbedienung

Bestimmte Handbücher für die Squeezebox und den Transporter enthalten einen Fehler im Abschnitt zu WPA-Kennwörtern. Laut Handbuch müssen Sie zum Einfügen eines Zeichens PLAY und zum Löschen eines Zeichens ADD drücken. Das Gegenteil ist der Fall: ADD fügt ein Zeichen ein und PLAY löscht eines.

Unterstützung von iTunes-Bibliotheken

Wenn Sie Apple iTunes zum Verwalten Ihrer Musik nutzen, importiert SqueezeCenter automatisch Daten aus Ihrer iTunes-Bibliothek. Wenn Sie Musik zu iTunes hinzufügen oder daraus entfernen, erkennt SqueezeCenter die Änderung und durchsucht die Bibliothek erneut. Sie können den Zeitraum zwischen den Prüfungen in der SqueezeCenter-Web-Benutzeroberfläche unter 'Einstellungen' auf der Registerkarte 'iTunes' festlegen.

Soll SqueezeCenter die iTunes-Datenbank nicht nutzen, müssen Sie die Einstellung in der SqueezeCenter-Web-Benutzeroberfläche unter 'Einstellungen' auf der Registerkarte 'iTunes' ändern. Sie müssen den Ordner angeben, in dem Ihre Musikdateien und Wiedergabelisten gespeichert sind, und dann auf der Registerkarte "Grundeinstellungen" auf "Musiksammlung durchsuchen" klicken.

Hinweis zu im iTunes Music Store erworbenen Titeln

Da Apple die meisten im iTunes Music Store erworbene Titel mit einem Kopierschutz (Digital Rights Management, DRM) versieht, sind die Benutzerrechte an diesen Dateien eingeschränkt. Ein wichtiges Recht, das von Apple nicht gewährt wird, ist die Wiedergabe auf einem nicht von Apple zur Verfügung gestellten Musikgerät (mit Ausnahme des Computers, auf dem der Titel erworben wurde). Erst wenn Apple seine Haltung gegenüber der Lizenzierung ändert oder FairPlay abschafft, können Sie diese Titel auf der Squeezebox oder dem Transporter wiedergeben.

Manche Titel aus der iTunes-Bibliothek werden in SqueezeCenter nicht angezeigt oder können nicht geöffnet werden

Wenn Sie Ihre iTunes-Musikdateien verschoben haben, exportiert iTunes unter Umständen falsche Informationen. Sie müssen sicherstellen, dass iTunes den richtigen Ordner für Ihre Musikbibliothek nutzt.

Klicken Sie in den iTunes-Einstellungen auf Erweitert. Ist der Pfad zu Ihrer Musikbibliothek falsch, müssen Sie ihn korrigieren. Ist er korrekt, wählen Sie einen anderen Ordner und anschließend wieder den korrekten Ordner. iTunes aktualisiert dann die Pfade zur gesamten Musiksammlung und exportiert diese Daten korrekt. Bei diesem Vorgang werden keine Musikdateien verschoben, sondern iTunes wird gezwungen, die Musikbibliothek zu durchsuchen und die Pfade zu den Titeln zu aktualisieren.

Bleibt das Problem weiterhin bestehen, wurden möglicherweise die iTunes-Einstellungen "iTunes Musik-Ordner verwalten" und "Dateinamen mit Titelnummer erstellen" geändert. Ändern Sie diese Einstellungen und stellen Sie anschließend den vorherigen Zustand wieder her, damit iTunes die aktuellen Informationen über Ihre Musikdateien aktualisiert. Beachten Sie, dass beim Ordnen der Musikdateien durch iTunes die Dateien selbst eventuell verschoben oder umbenannt werden.

Streaming-Wiedergabe über das Internet

SqueezeCenter wurde zur Übertragung von MP3-Musik auf eine Squeezebox, einen Transporter oder SLIMP3 Network Music Player entwickelt. Außerdem kann es dieselbe Musik über das Internet an einen beliebigen Software-Player, der MP3-Streams unterstützt, übertragen.

Installieren Sie zuerst SqueezeCenter und vergewissern Sie sich, dass der als Server zu nutzende Computer über das Internet erreichbar ist. Ist dies nicht der Fall, müssen Sie Anschluss 9000 auf dem Router zum SqueezeCenter-Computer öffnen (Anweisungen hierzu finden Sie in der Dokumentation zum Router). Dann:

  • Öffnen Sie folgenden Stream im Software-MP3-Player: http://localhost:9000/stream.mp3. Ersetzen Sie dabei "localhost" durch die IP-Adresse des SqueezeCenter-Computers. Wenn Sie SqueezeCenter durch ein Kennwort geschützt haben, müssen Sie die URL wie folgt anpassen: http://Benutzername:Kennwort@localhost:9000/stream.mp3. Bei bestimmten MP3-Playern werden Sie aufgefordert, Benutzername und Kennwort einzugeben.
  • Öffnen Sie die Web-Benutzeroberfläche von SqueezeCenter auf dem Remote-Computer, indem Sie die Seite http://localhost:9000 aufrufen (ersetzen Sie "localhost" durch die IP-Adresse des Remote-Computers). Es wird ein neuer "Player" entsprechend der IP-Adresse des Computers mit dem MP3-Software-Player angezeigt.
  • Suchen Sie auf der linken Seite der SqueezeCenter-Web-Benutzeroberfläche nach Musikdateien und wählen Sie die gewünschten aus. Ausgewählte Einträge werden auf der rechten Seite angezeigt.
  • Klicken Sie auf der rechten Seite der SqueezeCenter-Web-Benutzeroberfläche auf "Wiedergabe", um die Musik zu starten.
  • Nach wenigen Sekunden wird die Musik über den MP3-Software-Player ausgegeben. Die Verzögerung entsteht durch Puffern in der MP3-Player-Software.
  • Verwenden Sie die SqueezeCenter-Web-Benutzeroberfläche auf dem Remote-Computer, wenn Sie andere Inhalte wiedergeben möchten.
