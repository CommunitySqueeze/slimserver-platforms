Installazione

Per installare SqueezeCenter, fare doppio clic sul programma di installazione di SqueezeCenter e seguire le istruzioni visualizzate. Il programma di installazione visualizzerà un messaggio al termine della procedura. Per impostazione predefinita, SqueezeCenter per Windows viene installato come applicazione (per maggiori informazioni su come eseguire SqueezeCenter come servizio, fare clic qui). Fare doppio clic sull'icona di SqueezeCenter nell'area di notifica di Windows per avviare l'interfaccia Web di SqueezeCenter.

La prima volta che si esegue l'interfaccia Web una procedura guidata assisterà a configurare SqueezeCenter per il sistema.

Se si sta aggiornando a SqueezeCenter una versione esistente di SlimServer, le impostazione precedenti di SlimServer verranno mantenute. In seguito a modifiche apportate all'architettura dei plugin, tuttavia, sarà necessario scaricarne e installarne le versioni aggiornate. Le impostazioni specifiche dei plugin non verranno mantenute automaticamente. Per informazioni sui plugin, visitare il seguente sito web.

In SqueezeCenter sono disponibili informazioni su come utilizzare SqueezeCenter facendo clic sull'icona della guida situata nell'angolo inferiore sinistro dell'interfaccia Web.

Esecuzione di SqueezeCenter come servizio

Per impostazione predefinita SqueezeCenter viene eseguito ora in modalità applicazione e viene avviato quando si accede a Windows. Ciò consente a SqueezeCenter di accedere alla libreria di iTunes, alle cartelle condivise o alle unità di rete senza modificare alcuna attività. Per eseguire SqueezeCenter come servizio in background di Windows è necessario eseguire l'Attivatore dei servizi SqueezeCenter. Dal menu Start di Windows selezionare SqueezeCenter e quindi Gestione avvio.

  • Per avviare SqueezeCenter come servizio, selezionare Avvia SqueezeCenter automaticamente all'avvio del computer. SqueezeCenter verrà installato come servizio.
  • Per avviare SqueezeCenter come applicazione, selezionare Avvia SqueezeCenter quando si esegue l'accesso. SqueezeCenter verrà eseguito come applicazione (la configurazione predefinita).

Se si esegue SqueezeCenter come servizio in background è necessario immettere il nome utente e la password Windows per consentire a SqueezeCenter di accedere alla libreria di iTunes, alle cartelle condivise o alle unità di rete. Utilizzare il pulsante Avanti per accettare le modifiche e uscire dall'applicazione. Per completare le modiche è necessario riavviare Windows.

Aggiornamento firmware

Se si possiede un dispositivo Squeezebox o Transporter, al primo collegamento con SqueezeCenter si potrebbe ricevere una richiesta di aggiornamento del firmware. Tenere premuto il pulsante Brightness del telecomando fino a quando non inizia l'aggiornamento del firmware.

Immissione di una password WPA con il telecomando

In alcune versioni del manuale dell'utente di Squeezebox e Transporter è contenuto un errore nella sezione relativa alle password WPA. In base alle indicazioni fornite nel manuale è necessario utilizzare il pulsante PLAY per inserire un carattere nella stringa della password e ADD per eliminarlo. In realtà le due operazioni richiedono i tasti invertiti, ossia nel manuale dovrebbe essere indicato che ADD consente di inserire un carattere e PLAY di eliminarlo.

Supporto della libreria di iTunes

Se si utilizza iTunes di Apple per la gestione dei file musicali, SqueezeCenter importa automaticamente i dati della libreria di iTunes. Se si utilizza iTunes per aggiungere o rimuovere file musicali dalla relativa libreria, SqueezeCenter rileva la modifica e analizza nuovamente i dati dei brani. È possibile impostare l'intervallo di tempo fra le rilevazioni nella scheda iTunes di Impostazioni nell'interfaccia Web di SqueezeCenter.

Se non si desidera che SqueezeCenter utilizzi il database di iTunes è necessario cambiare l'impostazione nella scheda iTunes di Impostazioni nell'interfaccia Web di SqueezeCenter. Occorre specificare la cartella in cui sono memorizzati gli elenchi di riproduzione e la musica e quindi analizzare la musica selezionando Ripeti analisi nella scheda Impostazioni di base.

Considerazioni sui brani acquistati all'iTunes Store

Apple utilizza un wrapper DRM (Digital Rights Management) per la maggior parte dei brani venduti tramite l'iTunes Store; i diritti utente per tali file sono pertanto limitati. Un importante diritto che non viene concesso agli utenti è quello di riprodurre i brani su dispositivi musicali non forniti da Apple (l'unica eccezione è rappresentata dal computer con cui si acquistano i brani). Se la posizione di Apple resta invariata e non vengono quindi concesse licenze né viene eliminato FairPlay, non sarà possibile riprodurre questi brani su Squeezebox o Transporter.

Alcuni brani della libreria di iTunes non vengono visualizzati in SqueezeCenter o non è possibile aprirli

Se i file musicali presenti in iTunes sono stati spostati, è possibile che iTunes esporti informazioni errate sui brani della relativa libreria. È necessario accertarsi che in iTunes sia presente la cartella appropriata per la libreria musicale.

Accedere alle Preferenze di iTunes e fare clic su Avanzate. Se il percorso della cartella della libreria musicale è errato, è necessario correggerlo. Se è corretto, provare a specificare quello di una cartella diversa e quindi nuovamente quello corretto. In iTunes viene quindi eseguito l'aggiornamento dei percorsi di tutti i file musicali e i dati vengono esportati correttamente. Questa operazione non determina lo spostamento di alcun file musicale; viene semplicemente forzata l'analisi della libreria musicale e l'aggiornamento dei percorsi dei brani in iTunes.

Se il problema persiste, è possibile che l'eventuale modifica delle impostazioni Tieni organizzata la cartella iTunes Music o Crea il nome dei file con il numero della traccia generi confusione in iTunes. Se si desidera che iTunes aggiorni le informazioni corrette sui file musicali, provare a modificare queste impostazioni e quindi a ripristinarne lo stato precedente. Se si esegue l'organizzazione dei file musicali tramite iTunes, può essere necessario spostare o rinominare i file stessi.

Riproduzione di flusso tramite Internet

SqueezeCenter è stato progettato per riprodurre il flusso di file musicali MP3 in lettori musicali di rete Squeezebox, Transporter o SLIMP3. Oltre a ciò, consente anche di inviare il flusso audio della stessa musica in Internet a tutti i lettori software compatibili con il flusso MP3.

In primo luogo, installare SqueezeCenter e assicurarsi che il computer da utilizzare come server sia raggiungibile in Internet. In caso contrario, è necessario inoltrare la porta 9000 del router al computer in cui viene eseguito SqueezeCenter. Consultare la documentazione del router per istruzioni. Procedere quindi come segue:

  • Utilizzare il software del lettore MP3 per aprire un flusso nell'URL: http://hostlocale:9000/stream.mp3. (sostituire "hostlocale" con l'indirizzo IP del computer in cui è in esecuzione SqueezeCenter). Se per SqueezeCenter si utilizza una password di sicurezza è necessario modificare l'URL nel modo seguente: http://nomeutente:password@hostlocale:9000/stream.mp3. In alcuni lettori viene richiesta l'immissione di nome utente e password.
  • Avviare l'interfaccia Web di SqueezeCenter in esecuzione sul computer remoto aprendo la pagina Web http://hostlocale:9000 (sostituire "hostlocale" con l'indirizzo IP del computer remoto). Si noterà un nuovo lettore corrispondente all'indirizzo IP del computer in cui è installato il lettore software MP3.
  • Utilizzare il riquadro di sinistra dell'interfaccia Web di SqueezeCenter per sfogliare e selezionare la musica da riprodurre. Una volta selezionati, i brani musicali vengono visualizzati nel riquadro di destra dell'interfaccia Web.
  • Fare clic sul pulsante Play nel riquadro di destra dell'interfaccia Web di SqueezeCenter per avviare la riproduzione dei brani musicali.
  • Dopo qualche secondo verrà avviata la riproduzione della musica tramite il lettore software MP3. Il ritardo è dovuto al buffering da parte del lettore software.
  • Per cambiare il brano riprodotto, utilizzare l'interfaccia Web di SqueezeCenter nel computer remoto.
